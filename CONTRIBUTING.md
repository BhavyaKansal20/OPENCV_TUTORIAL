# 🤝 Contributing Guidelines

Thank you for your interest in contributing 🚀
This project is maintained by **Bhavya Kansal** and focuses on real-world AI applications.

---

## 🧠 Contribution Scope

You can contribute by:

* Improving model performance or efficiency
* Enhancing UI/UX or deployment
* Fixing bugs or issues
* Improving documentation
* Suggesting better approaches

---

## 🛠 How to Contribute

1. Fork the repository
2. Create a new branch:

```bash
git checkout -b feature/your-feature-name
```

3. Make your changes
4. Commit your changes:

```bash
git commit -m "feat: improve model performance"
```

5. Push to the branch:

```bash
git push origin feature/your-feature-name
```

6. Open a **Pull Request** and describe the changes you've made.

---

## 🛑 Code Standards

* Ensure your code is clean and properly formatted.
* Do not expose any API keys or secrets in your commits.
* Make sure your PR resolves an existing issue or adds a clearly defined feature.

---

Thank you for helping to improve AAGNI AI! 🔥
